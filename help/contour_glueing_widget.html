Contour Glueing Widget
Leif Laaksonen CSC 2000
 **************************************************************************

Through this widget it is possible to "glue" a particulat plt file to a particular structure file. In some cases you might have several structures and several plt files read into gOpenMol. The "glueing" controls which plt file should be connected to which structure.

Select the structure from the list by clicking at the option menu button. The widget says at the top which plt file should be glued.

To make the selection click at the "Apply" button.

After click at athe "Apply" button the new selection will appear in the main Contour widget.
