Contour Plane Animate Widget
Leif Laaksonen CSC 2001
 **************************************************************************

Control the smooth display of the contour plane. The default number of steps is the number of points in the selected direction of the grid data. It is possible to define a new value for the number of steps along the axis. The new step length will be used when any of the control buttons is pressed.

Cut plane display control:

  • |<... go to Min x, y or z value
  • <... back one step
  • []... halt the display --->+
  • |>... play forward one step at the time
  • >... forward one step
  • >|... go to Max x, y or z value

To scale the display differently define new values for the Min and Max values in the contour plane.

Line command: see plot command
