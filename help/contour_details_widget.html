Contour Details Manager Widget
Leif Laaksonen CSC 1996
 **************************************************************************

This is the contour details display widget. Through this widget you can control the contour details: Surface smoothing, surface type (solid/mesh), transparency and display state (on/off).

Use the radio buttons to choose the property and use the slider to define the transparency of the selected isocontour value.

Surface details:

  • Surface smoothing.
  • Surface transparency.
  • Surface type mesh/solid.
  • Surface display state.

After the selection remember to press the "Apply" button.

Line command: see contour command
