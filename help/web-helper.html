Using gOpenMol as a Web helper application

For Windows

Do the following:

  1. Open Windows Explorer
  2. Open view - Options
  3. Click File Types
  4. Add new type by clicking the New Type button
  5. Add to Description of type: gOpenMol gom file Content Type (MIME): chemical/x-gom Default extension: .gom
  6. Action: Edit the "open" action in the "Application used to perform action:" to point to the rungOpenMol.bat file. Example: "D:\USERS\dummy\files\gopenmol\bin\rungOpenMol.bat" "%1"
  7. Click OK
  8. Click Close
  9. Click Close

For Unix

Do the following:

Open the mailcap file in your home directory and and the following line to the file:

chemical/x-gom; /your/gopenmol/directory/gopenmol/bin/rungOpenMol %s

where /your/gopenmol/directory/gopenmol/bin is the directory where you have installed gOpenMol.

You can try downloading gom files from the pages:

http://laaksonen.csc.fi/gopenmol/gomodel.html
and
http://www.fos.su.se/physical/sasha/dna_sim/
