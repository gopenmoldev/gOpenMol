#
# edit atom bonding
#

edit  bond  brea$k   Seg1 Res1 Atm1 
                     Seg1 Res1 Atm1 Seg2 Res2 Atm2
            crea$te  Seg1 Res1 Atm1 Seg2 Res2 Atm2

#
# edit hydrogen bonding
#

edit  hbond  brea$k   Seg1 Res1 Atm1 
                      Seg1 Res1 Atm1 Seg2 Res2 Atm2
             crea$te  Seg1 Res1 Atm1 Seg2 Res2 Atm2

#
# Set hydrogen bond search subset.
#

edit  hbsu$bset       Seg  Res  Atm

# Reset hydrogen bond search subset.
#

edit  -hbs$ubset
